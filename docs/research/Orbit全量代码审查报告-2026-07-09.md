### 3.2 P1 严重（影响核心功能）

#### P1-1: `resource_guard` async/await 未同步

**文件**: `src/orbit/resource_guard/resource_guard.py`
**影响**: 10+ 测试因 `GuardDecision` 返回协程而非值而失败

```python
# 当前（错误）
async def guard_request(...):
    return GuardDecision.ALLOW  # 直接返回

# 调用方
result = guard_request(...)  # 返回协程对象，无 .decision 属性
```

**测试失败**:
```
FAILED test_resource_guard.py::TestResourceGuard::test_guard_request_allow
  → AttributeError: 'coroutine' object has no attribute 'decision'
```

**修复方案**: 确保所有 `guard_request()` 调用使用 `await`

---

#### P1-2: `analyzer` 函数签名变更未同步测试

**文件**: `tests/unit/test_analyzer.py`
**根因**: `_detect_python_framework()` 新增 `rel_paths` 参数，测试仍用旧签名

```python
# 实际签名（变更后）
def _detect_python_framework(files: list[str], rel_paths: list[str] = []) -> str:

# 测试调用（未更新）
assert "Flask" in _detect_python_framework(["flask==2.0"])
#                                     ↑
#                             缺少 rel_paths 参数
```

**影响**: 7 个测试全部失败

### 3.3 P2 警告（影响鲁棒性）

#### P2-1: offpeak_scheduler Windows tzdata 依赖

**文件**: `src/orbit/scheduler/offpeak/scheduler.py`
**问题**: `ZoneInfo("Asia/Shanghai")` 在 Windows 上需要 `tzdata` 模块

```
ModuleNotFoundError: No module named 'tzdata'
zoneinfo._common.ZoneInfoNotFoundError: 'No time zone found with key Asia/Shanghai'
```

**影响**: 10+ 时区相关测试在 Windows CI 失败
**建议**: 添加 `tzdata` 到依赖，或回退到 `pytz`
