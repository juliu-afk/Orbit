### 2.5 防幻觉 guard 抽象（P2）

**当前**：5 个文件各有 3 段重复 guard

**方案**：在 `hallucination/` 包内建 `base.py`：

```python
# hallucination/base.py
from functools import wraps
from orbit.hallucination import ValidationResult

def skip_if_empty(func):
    """代码为空时跳过验证"""
    @wraps(func)
    def wrapper(self, code: str, *args, **kwargs):
        if not code.strip():
            return ValidationResult(
                passed=True,
                level=self.level,  # 子类定义
                warnings=["empty code, skipped"],
            )
        return func(self, code, *args, **kwargs)
    return wrapper

def skip_if_no_sandbox(func):
    """沙箱不可用时跳过验证"""
    @wraps(func)
    async def wrapper(self, code: str, *args, **kwargs):
        sandbox = getattr(self, '_sandbox', None)
        if sandbox and not await sandbox.is_available():
            return ValidationResult(
                passed=True,
                level=self.level,
                warnings=["sandbox unavailable, Lx skipped"],
            )
        return await func(self, code, *args, **kwargs)
    return wrapper

class BaseValidator:
    """所有防幻觉层基类"""
    level: HallucinationLevel  # 子类定义

    def __init__(self, sandbox=None):
        self._sandbox = sandbox
```

**各层改造**（以 l1_graph.py 为例）：

```python
# 改前
```

## 三、数据流

```
PRD 验收标准
    ↓
技术方案（本文）
    ↓
各方案独立 PR
    ↓
CI：全量回归 + 覆盖率门禁
    ↓
用户审查 + merge
```

每项改动走独立 feature 分支。原因：
- 单 PR 出问题独立回滚，不污染其他
- 审查聚焦，降低认知负荷
