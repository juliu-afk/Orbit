## 问题列表

### P2：system_prompt 未传递到多模态调用（建议修复）

- **位置**：`client.py:_generate_multimodal` 第 138-142 行
- **问题**：从 `req.messages` 提取了 `system_msg` 但未传递给 `VisionAdapter.generate()`
- **影响**：多模态模型调用不携带 Orbit 的编排层 system_prompt
- **严重程度**：轻微——视觉理解任务不需要 agent 行为约束
- **建议**：P1 在 VisionAdapter 中支持 `system_prompt` 参数

### P3：content 中 system role 未被正确处理（已知限制）

- **位置**：`vision.py:_generate_with_downgrade` 第 103 行
- **问题**：messages 的 content 数组直接发送，不处理 system role 分离
- **影响**：如果 Agent 在多轮对话中传入 system role 的 content block，API 可能拒绝
- **严重程度**：极低——P0 只支持单轮多模态调用
- **建议**：P1 支持多轮多模态对话时处理
