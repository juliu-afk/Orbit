## Mock 策略

| 依赖类型 | Mock 方式 |
|---------|----------|
| LiteLLM / LLM 调用 | `unittest.mock.patch` 替代网关 |
| Docker 沙箱 | `patch('orbit.sandbox.executor.ProcessSandbox')` |
| 文件系统 I/O | `tmp_path` fixture + `patch('builtins.open')` |
| GUI 截图 (mss/pyautogui) | `patch('mss.mss')`, `patch('pyautogui.screenshot')` |
| OCR 模型 | `patch` 模型加载函数返回 mock |
| BGE 向量 | `patch('sentence_transformers.SentenceTransformer')` |
| 数据库 | 已有 `conftest.py` fixtures |
