## 工具设计

### 图谱查询类（复用 CodeNode + edges）

| 工具 | 输入 | 数据源 | 输出 |
|------|------|--------|------|
| `trace_path` | symbol, direction, max_depth | edges 表 BFS | {callers:[], callees:[]} |
| `get_architecture` | 无 | CodeNode + edges 聚合 | {languages, modules, entry_points, hotspots} |
| `search_code` | pattern, file_pattern | CodeNode.file_path + grep | [{file, line, content, symbol}] |
| `dead_code` | 无 | edges 反查 + 白名单 | [{name, file_path, type}] |
| `find_implementations` | class_name | edges(INHERITS) + ast bases | [{name, file_path}] |

### 语义编辑类（CodeNode 定位 + 文件读写）

| 工具 | 输入 | 核心逻辑 |
|------|------|---------|
| `replace_symbol_body` | symbol, new_body | CodeNode.start_line..end_line → read → replace → write → incremental_update |
| `insert_after_symbol` | symbol, code | end_line+1 → insert → write → incremental_update |
| `insert_before_symbol` | symbol, code | start_line → insert → write → incremental_update |
| `safe_delete_symbol` | symbol | edges 查入度=0 → read+delete_lines → write → delete CodeNode+edges |

## API 设计

所有工具通过 MCP JSON-RPC 暴露，不新增 REST 端点。

```json
// trace_path 示例
{"method":"tools/call","params":{"name":"trace_path","arguments":{"symbol":"ProcessOrder","direction":"both","max_depth":3}}}
// → {"callers":[{"name":"handleRequest","file":"api.py","line":42},...],"callees":[...]}

// replace_symbol_body 示例
{"method":"tools/call","params":{"name":"replace_symbol_body","arguments":{"symbol":"deprecated_func","new_body":"    raise NotImplementedError('Use new_func instead')"}}}
// → {"success":true,"symbol":"deprecated_func","file":"utils.py","old_lines":"20-25"}
```

## 风险点

| 风险 | 缓解 |
|------|------|
| 编辑工具写坏文件 | 写前备份原内容到内存，写后验证文件仍可 AST 解析 |
| trace_path 大图 BFS 超时 | depth 上限 5，每层最多 100 节点截断 |
| OKF 导出覆盖已有 bundle | 仅覆盖同名 concept，不删未知文件 |
| Serena 清理遗漏 | `rg -i serena src/ tests/ configs/` 零结果门禁 |
