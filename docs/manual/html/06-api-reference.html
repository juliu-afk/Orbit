6.1 统一约定 · Conventions

  • 版本前缀：/api/v1（API_V1_STR）。
  • 鉴权：设置 ORBIT_AUTH_TOKEN 后启用 AuthMiddleware；请求带 Authorization: Bearer &lt;token&gt;。
  • 自动文档：/docs（Swagger UI）、/redoc（ReDoc）。
  • 指标：/metrics（Prometheus）。
