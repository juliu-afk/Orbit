2.3 为什么 GLM-4.6V 是游戏规则改变者

  • 2025.12.8 发布——最新一代，比 GLM-4.5V 晚 4 个月
  • Flash 版完全免费+MIT 开源——零成本，可商用，9B 模型超 Qwen3-VL-8B
  • 原生多模态 Function Call——视觉感知→可执行行动，全链路打通。这对 Computer Use 是质变
  • 128K 上下文——可处理 150 页文档、200 页 PPT、1 小时视频
  • API 降价 50%（vs GLM-4.5V）——输入 1 元/百万 tokens
  • 与 Orbit 已有集成零摩擦——智谱 Coding Plan 的 api_base 和 api_key 已在 Orbit Gateway 中配置
