### 设计原则（来自 mattpocock/skills）

| 原则 | 含义 | Orbit 现状 |
|------|------|-----------|
| Process > Prompt | 定义工作流，不是定义输出 | ✅ 状态机驱动 |
| Progressive Disclosure | 根文件 ≤50 行，细节按需加载 | ❌ TaskContext 全量加载 |
| Depth-first | 一个决策分支穷尽再开下一个 | ❌ Clarifier 未结构化 |
| Explore before ask | 代码库能回答的不问用户 | ✅ SCOPING 扫描器 |
| Self-extending | 系统能生成自己的技能 | ❌ 手动编写 Agent 类 |
