🎬 10. yt-dlp + ffmpeg 管线复用

复用的（独立于模型）

yt-dlp 下载器覆盖 1800+ 平台——没有一个国产模型能替代。Orbit 必须集成。
ffmpeg 抽帧 + 去重审计截图管线；无视频能力的模型（如 DeepSeek-VL2）作为 fallback 时需要。
字幕优先策略先用免费原生字幕，没有才走 ASR。降本通用思路。
帧预算算法≤30s→30帧，10min+→100帧封顶。token 控制优秀。

不需要的（国产原生支持更好）

Whisper 转录→ GLM-4.1V / Qwen3.5-Omni 原生理解视频音频
第三方多模态调用→ Qwen3.7-Plus / GLM-4.1V，OpenAI 兼容
外部插件机制→ Orbit Tool Registry + OpenAI 直连
ffmpeg 视频预处理→ GLM / Qwen 原生支持 2h 视频直传，跳过抽帧
