## 偏差说明

1. **路由层推迟**: 路由文件（chat/git_routes/codegraph_routes）依赖 git subprocess/SQLite/WebSocket，mock 代价高。改为优先高收益纯函数模块。
2. **95% 目标**: 每轮新增约 0.5-1% 覆盖率。从 69%→72%（+3%）。要达 95% 还需约 30-40 个测试文件，需多轮迭代。
3. **circuit breaker xfail**: LLM 全部失败时任务状态为 DONE 而非 FAILED——调度器/Agent 错误处理逻辑可能已变更，需独立排查，不属于测试修复范围。
