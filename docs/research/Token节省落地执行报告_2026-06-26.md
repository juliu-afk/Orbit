## 6. Token 节省规则

6 条规则已写入三项目 CLAUDE.md：

| # | 规则 | 机制 |
|---|------|------|
| 1 | 按需加载文档 | WORKFLOW.md 仅在收到新需求时读 |
| 2 | Grep/Glob 优先 | 避免 bash 文件读取 |
| 3 | Subagent 用 cavecrew | 输出压缩 ~60% |
| 4 | 测试输出截断 | 通过时不读，失败时只读最后 30 行 |
| 5 | Diff 先于全量测试 | 不盲目全量回归 |
| 6 | Memory 系统 | 重复问题自动注入 |
