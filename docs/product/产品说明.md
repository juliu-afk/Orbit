# Orbit 产品说明

> 版本 0.12 | 2026-06-28

---

## 一、Orbit 是什么

Orbit 是一个**AI 驱动的软件开发自动化系统**。你只需要用自然语言描述需求，Orbit 就能自动分析、设计、编写代码、测试代码，最后把结果报告给你。

打个比方：**如果传统开发是"自己开车"，Orbit 就是"告诉司机去哪——司机会自己认路、自己开、自己检查车况"。**

---

## 二、解决了什么问题

| 问题 | Orbit 怎么做 |
|------|-------------|
| 写代码太慢 | AI 自动写，人只需审核 |
| 项目交接成本高 | AI 记住所有代码和决策，新成员直接问 |
| 重复劳动多 | 测试、格式化、审查——AI 自动完成 |
| 团队经验流失 | 记忆系统记住每次成功和失败，下次更聪明 |
| 跨项目知识复用 | 从一个项目学到的经验，自动迁移到新项目 |

---

## 三、核心功能

### 3.1 自然语言编程
你跟 Orbit 说"帮我做一个登录功能"，它就会：
1. 分析你的需求
2. 设计实现方案
3. 编写前端+后端代码
4. 自动运行测试
5. 展示结果

### 3.2 多 Agent 协作
Orbit 内部有多个 AI"专家"分工合作：
- **架构师**：负责设计方案
- **开发者**：负责写代码
- **审查员**：负责检查代码质量
- **测试员**：负责验证功能

他们自动沟通、自动协作，不需要你协调。

### 3.3 /Compose 编排模式
写一份 `spec`（任务说明书），Orbit 自动拆解成多个子任务，派给不同的 Agent 并行执行，最后汇总结果。适合大功能开发。

### 3.4 持续记忆
Orbit 会记住你项目的每一次决策、每一次错误、每一次成功。用得越久越聪明——它会记住你的偏好、你的代码风格、你常遇到的问题。

### 3.5 智能安全
- **权限系统**：Agent 不能随便删文件、改配置
- **防幻觉**：AI 输出会被多层验证，减少"胡编乱造"
- **审计日志**：所有操作可追溯
