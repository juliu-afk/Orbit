## 2. 关键设计

### A4-A6: 工具安全三层钩子（ADR-6）

```
registry.dispatch()
  → PermissionEngine.check(role, tool_name)    ← A4
    → shell.exec_command()
      → BashValidators.validate(command)        ← A5
    → filesystem.write_file/edit_file
      → WorkspaceGuard.validate(path)           ← A6
```

### A2-A3: 全局实例生命周期

```python
# main.py 模块级（app 创建前）
_actor_registry = ActorRegistry()
_actor_spawn = ActorSpawn(registry=_actor_registry, agent_factory=AgentFactory)
_compose_orch = ComposeOrchestrator(actor_spawn=_actor_spawn)
_permission = PermissionEngine()
_watchdog = ActorWatchdog(_actor_registry)

# app startup:
asyncio.create_task(_watchdog.run())
```

### A7: Compose 端点

```
POST /api/v1/compose/run
  Body: {"spec": "title: ...\ntasks:\n  - id: ..."}
  → ComposeOrchestrator.run_spec(spec_text)
  → {"code": 0, "data": {"status": "ok", "tasks": {...}}, "message": "ok"}
```
