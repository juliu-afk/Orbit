## 核心概念速查

- **六图谱**：代码图谱（Tree-sitter）+ 数据库图谱（DDL/MVCC）+ 配置图谱（.env/Nginx/docker-compose）+ 知识图谱（外挂领域知识）+ 元图谱（跨图谱关系）+ 文档图谱
- **9 层防幻觉**（2026-07-08 按 `src/orbit/hallucination/` 代码校正）：pipeline 实跑 **L1-L8**，顺序 L1→L4→L3→L2→L6→L8→L7→L5 —— L1 图谱校验 / L2 动态追踪 / L3 熵监控 / L4 类型检查 / L5 Z3 形式化 / L6 合约验证 / L7 运行时沙箱 / L8 配置漂移；**L9 动态合规**住在 `compliance/` 独立模块（不在 pipeline 内）
- **熔断**：Token 计数器 + 延迟阈值，超限即熔断回滚到上一检查点
- **50+ 个源码模块**：`src/orbit/` 下已落地的全部模块（2026-07-10，含 Phase 1-3 新增 6 模块：change_detector/query/artifact/watcher/okf_importer/okf_exporter）
