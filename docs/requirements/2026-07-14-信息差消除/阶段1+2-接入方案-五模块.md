## 2. 接入方案

### 2.1 P0 偏离日志 — 接到 OrbitWiring

**接入点**: `integration/wiring.py:OrbitWiring`

```python
# on_task_start() — 创建 DeviationLogger 实例
self._deviation_loggers[task_id] = DeviationLogger(task_id=task_id)

# on_task_end() — flush + feed evolution
dev_log = self._deviation_loggers.pop(task_id, None)
if dev_log:
    await dev_log.flush_to_checkpoint(self._ckpt_manager)
    scope = self._get_scope()
    if scope:
        scope.add_deviations_batch(task_id, dev_log.to_tactical_rules())
    gepa = self._get_gepa()
    if gepa:
        await gepa.evolve_from_deviations(dev_log.get_failure_reasons())
```

**接入点2**: `agents/base.py` 或 loop — Agent 决策分叉处调用 `wiring.record_deviation()`

```python
# Agent 发现偏离时
wiring = get_wiring()
wiring.record_deviation(task_id, planned, actual, reason, ...)
```
