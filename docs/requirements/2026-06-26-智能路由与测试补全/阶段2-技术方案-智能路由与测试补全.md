class ModelTier(StrEnum):

@dataclass

### 2.3 RouterAgent

```python
class RouterAgent:
    """PLANNING 阶段评估任务复杂度，输出推荐模型级别。"""

    def __init__(self, weights: ScoreWeights | None = None):
        self.weights = weights or ScoreWeights.from_env()  # 支持环境变量动态调整

    async def evaluate(self, task_context: TaskContext) -> RouterDecision:
        """输入 TaskContext，输出模型推荐。"""
```

评分维度与权重（默认值，可通过环境变量 `ROUTER_WEIGHT_*` 覆盖）：

| 维度 | 权重 | 环境变量 | 判定逻辑 |
|------|------|---------|---------|
| 涉及文件数 | 30 | `ROUTER_WEIGHT_FILES` | 1→0分, 2-5→50分, 6+→100分 |
| 修改类型 | 25 | `ROUTER_WEIGHT_CHANGE` | 配置→0分, 单函数→30分, 多模块→100分 |
| 风险等级 | 25 | `ROUTER_WEIGHT_RISK` | 无风险→0分, 高风险(支付/安全/并发)→100分 |
| Agent 角色 | 15 | `ROUTER_WEIGHT_ROLE` | ConfigAgent→0分, Developer→50分, Architect→100分 |
| 历史相似 | 5 | `ROUTER_WEIGHT_HISTORY` | 有成功先例→-30分(降级) |

Tier 判定阈值：

| 总分 | Tier |
|------|------|
| 0-20 | Tier 0（本地规则） |
| 21-50 | Tier 1（DS Flash） |
| 51-75 | Tier 2（GLM-5.2） |
| 76-100 | Tier 3（DS V4 Pro） |

class Scheduler:
    def __init__(self, ..., router: RouterAgent):  # 新增
        self.router = router

    async def _agent_cycle(self, task_id, state, context):
        # PLANNING 阶段：调用 RouterAgent
        if state == TaskState.PLANNING:
            task_ctx = self._build_context(task_id, context)
            decision = await self.router.evaluate(task_ctx)
            context["model_tier"] = decision.tier.value
            context["router_decision"] = decision

## 6. 数据流

```
用户提交任务
  → Scheduler.run_task()
    → PLANNING 阶段
      → RouterAgent.evaluate(task_context)
        → 复杂度评分（5 维度加权）
        → 输出 RouterDecision（tier + confidence）
      → AgentModelResolver.resolve(agent_name, decision)
        → 检查 CC_SWITCH → 环境变量 → Router 推荐 → 默认
        → 输出 ResolvedModel（model + source）
    → CODING 阶段
      → LLMClient.generate(req, task_id, agent_name, decision)
        → resolver.resolve() 获取实际模型
        → 调用 LiteLLM
        → 审计表记录 actual_model + model_source
        → 失败 → 降级升级
```

## 8. 风险点

| 风险 | 缓解 |
|------|------|
| RouterAgent 评分不准导致错误降级 | 权重可配置 + CI 中验证决策准确率 |
| CC_SWITCH 格式解析错误导致模型选择失败 | 解析失败→WARN→降级到下一个优先级 |
| Resolver 增加延迟影响 PLANNING 阶段性能 | Resolver 纯内存操作，无 I/O，<1ms |
| 审计表 DDL 变更影响现有数据 | ALTER TABLE ADD COLUMN，可重复执行 |
