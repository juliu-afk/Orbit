📋 必要性说明

🔴 不做

  • Agent 盲视——80% 的软件信息是视觉的
  • Bug 诊断低效——用户只能文字描述
  • Computer Use 缺失——无法操作真实 UI
  • 竞品降维打击——Codex/Cursor 已集成视觉

🟢 做了

  • 差异化护城河——唯一自研 CU 的多 Agent 系统
  • Bug 诊断提效 10×——录屏→自动定位代码
  • 审计合规——每步决策有截图证据
  • 国产自主可控——零外国闭源 API 依赖
