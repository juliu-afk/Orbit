## 2. API 设计

### 2.1 SSE 流式端点加 token 认证

```
GET /api/v1/agent/{agent_id}/stream?task_id=...&token=...
```

**变更**：`dependencies.py` 新增依赖项

```python
async def verify_stream_token(token: str = Query(...)) -> str:
    """校验流式端点 token——与 WebSocket token 同源。"""
    payload = decode_token(token)  # 复用现有 JWT
    if payload.get("scope") not in ("agent:stream", "agent:*"):
        raise HTTPException(403, "token scope 不足")
    return payload["sub"]
```

**路由层**：`routes/agent.py` 已有 `stream` 端点，加 `_token: str = Depends(verify_stream_token)`。

### 2.2 错误码

| 端点 | 状态 | code | message |
|-----|:--:|------|-----|
| SSE stream | 403 | 40301 | token scope 不足 |
| SSE stream | 401 | 40101 | token 过期或无效 |
| Agent run | 404 | 40401 | agent_id 不存在 |

## 3. 数据模型

### 3.1 FTS5 BM25 评分——SQL 变更

```sql
-- memory/store.py search 方法改用
SELECT path, snippet(body, 0, '<b>', '</b>', '...', 40) AS snippet,
       bm25(memory_fts, 0.0, 1.0, 0.75) AS score  -- k1=1.0, b=0.75
FROM memory_fts
WHERE memory_fts MATCH :query
ORDER BY score DESC
LIMIT :limit;
```

FTS5 默认不提供 `bm25()`——需配置 `rank='bm25'` 参数。实际实现：

```python
cursor.execute(
    "SELECT ... FROM memory_fts WHERE memory_fts MATCH ? "
    "ORDER BY rank LIMIT ?",
    (query, limit)
)
conn.create_function("bm25", 1, _bm25_score)  # Python 侧实现
```

### 3.2 /dream 定时器——文件时间戳

```python
# dream/scheduler.py
DREAM_INTERVAL_DAYS = 7
LAST_DREAM_FILE = ".orbit/last_dream"

async def should_dream(self) -> bool:
    try:
        mtime = os.path.getmtime(self.LAST_DREAM_FILE)
        elapsed = (time.time() - mtime) / 86400
        return elapsed >= self.DREAM_INTERVAL_DAYS
    except FileNotFoundError:
        return True  # 从未运行过

async def mark_dream_done(self):
    Path(self.LAST_DREAM_FILE).write_text(
        datetime.now(UTC).isoformat()
    )
```

## 9. 边界 case 清单

| 场景 | 预期行为 |
|------|---------|
| **ChatPanel 无 lastTaskId** | ChatStream 不渲染，无报错 |
| **SSE 连接中断（5A.3）** | finally 块 cancel token，EventSource 自动重连 5 次 |
| **SSE 连接中断（5A.4）** | ChatStream `useEventSource` 自动重连，3 次失败后 emit("error") |
| **token 参数缺失/无效** | 403 / 401，EventSource onerror 触发 |
| **tiktoken 未安装** | `TokenCounter` 回退 chars/4 |
| **jieba 未安装** | `_tokenize_cjk()` 回退 Unicode `\w+` 正则 |
| **/dream 首次运行** | `.orbit/last_dream` 不存在 → 执行 dream |
| **/dream 时钟回拨** | mtime > now → 跳过本次，记录 warning |
| **FTS5 BM25 不可用** | 回退 `ORDER BY rank` (默认 BM25) |
| **Compose 子任务失败 ≤2 个** | retry(2) → 仍失败 → partial_success |
| **Compose 子任务全部失败** | 整体 FAILED，不标记 partial |
| **Actor 5 分钟无心跳** | stale → reap → FAILED |
| **Actor 心跳恢复（假阳性）** | reap 前检查最后心跳，10 秒容差 |

## 10. 风险与缓解

| 风险 | 概率 | 影响 | 缓解 |
|------|:--:|:--:|------|
| tiktoken 模型名不匹配 o200k_base | 中 | token 估算误差 >10% | 启动时检测模型，自动选编码器 |
| ChatPanel→ChatStream 串联导致 Vue 重渲染 | 低 | 消息闪烁 | `v-if` 仅在 `lastTaskId` 变更时触发 |
| jieba 分词性能（14MB 词典加载） | 低 | 首次查询慢 200ms | lazy import + 单例 warmup |
| Actor stale 误杀 | 中 | 活跃任务被终止 | 10 秒容差 + reap 前重读心跳 |
| SSE finally 与 CancellationToken 竞态 | 低 | 重复 cancel | `cancel_token.cancel()` 幂等 |
