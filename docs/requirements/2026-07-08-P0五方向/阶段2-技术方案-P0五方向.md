## 二、方向 2：Bandit 路由

### 2.1 新文件：`router/bandit.py`

```python
class ThompsonBandit:
    """Thompson Sampling 多臂 Bandit——Beta-Bernoulli 共轭。

    每个模型臂维护 Beta(α, β) 后验。
    决策: 采样各臂→选最大采样值。
    更新: 成功→α+1, 失败→β+1, 延迟>3s→β+0.5(软惩罚)。
    """
    def __init__(self, arms: list[str], prior_alpha=1.0, prior_beta=1.0)
    def select(self) -> str
    def update(self, arm: str, success: bool, latency_ms: float = 0)
    @property def posteriors(self) -> dict[str, dict[str, float]]
```

### 2.2 修改：`router/agent.py`

- `RouterAgent.__init__` 新增 `bandit: ThompsonBandit | None = None`
- `RouterAgent.evaluate()` 末尾：若 `self._bandit` 存在且 `ORBIT_ROUTER_BANDIT=1`，用 bandit 推荐的 tier 覆盖固定权重 tier
- Bandit 更新由 task runner 在任务完成后调用（新增 `agent.update_bandit(task_id, success, latency)` ）

### 2.3 环境变量

```python
class ConformalPredictor:
    """Inductive Conformal Prediction——distribution-free 置信预测集。

    校准: 历史 (task, code, outcome) → 非一致性评分分布
    预测: 新任务+N候选 → 1-α 置信预测集
    """
    def __init__(self, alpha=0.05)
    def calibrate(self, calibration_data: list[tuple[str, str, bool]])
    def predict(self, task: str, candidates: list[str]) -> list[str]
    def _nonconformity(self, task, code, success) -> float
```

```python
class CUSUMDriftDetector:
    """CUSUM 变点检测——监控模型输出分布急变。

    独立监控 (success_rate, latency, output_len) 三元组。
    任一 metric CUSUM > h=5 → DriftAlert。
    """
```
