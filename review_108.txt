## 🟡 P1

### P1-1: `missing-permission` 规则误报率 100%

```python
("missing-permission", "API 端点缺少 @require_permission 装饰器", "warning",
 r"@router\.(?:get|post|put|delete|patch)\("),
```

正则匹配**所有**路由装饰器，而非检查装饰器是否**缺失**。每个有 `@router.get(...)` 的行都会被标记为 `missing-permission`，无论是否已有 `@require_permission`。

**修复**: 改为正向检查——扫描文件收集所有路由端点，再检查每个端点的前面/上面是否有 `@require_permission` 行。

---

### P1-2: ThemeToggle 图标字符串未导入，可能渲染空白

```vue
<el-button :icon="isDark ? 'Sunny' : 'Moon'">
```

Element Plus 的 `el-button` `icon` prop 需要导入的组件引用，不是字符串：

**修复**:
```typescript
import { Sunny, Moon } from '@element-plus/icons-vue'
// template: :icon="isDark ? Sunny : Moon"
```
