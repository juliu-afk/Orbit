## TaskState 状态机（MVP 阶段）

```
IDLE → (创建) → IDLE → (取消) → CANCELLED
                                   ↓
                        后续 Step 5.x 接入完整状态转换：
                        IDLE → PARSING → PLANNING → CODING → VERIFYING → DONE/FAILED
```
