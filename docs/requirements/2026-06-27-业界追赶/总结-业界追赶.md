## 5. 全链路可跑通路径

```
POST /api/v1/compose/run  (AC-A7)
  │
  ├─ ComposeParser 解析 spec YAML
  ├─ _topological_sort() Kahn 拓扑排序
  ├─ _spec_review() 方案审查门禁
  │
  ├─ WorktreeManager.create()   ← AC-B4 隔离工作区
  ├─ ActorSpawn.spawn() ×N      ← AC-B2 并行子Agent
  │     │
  │     ├─ AgentFactory.create() → ReActAgent
  │     ├─ PermissionEngine.check()  ← AC-A4 权限前置
  │     ├─ BashValidators.validate() ← AC-A5 Shell白名单
  │     ├─ WorkspaceGuard.validate() ← AC-A6 路径守卫
  │     │
  │     └─ execute_stream()     ← AC-19 流式循环
  │           ├─ LLM思考 (think)
  │           ├─ 工具调用 (act)
  │           ├─ GoalJudge.evaluate() ← AC-B1 目标自检
  │           └─ yield StreamEvent  ← AC-19 SSE推送
  │
  ├─ _code_review() 代码审查门禁
  └─ return {"status": "ok", ...}
```
